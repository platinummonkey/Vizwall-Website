VIZLAB GUIDELINES

The calendar request form is to serve for coordination of all activities associated with the Advanced Visualization Laboratory (VizLab). This process will allow the VizLab team to better serve the UTSA community and other collaborative efforts engaging in this resource.

  1. Requests for a demonstration or usage of the facility should be made via the calendar request form.
  2. Requests generally should be done 72 hours in advance
  3. Requested times should be Monday through Friday between the hours of 8am - 5pm.
  4. Data files to be used on the visualization wall should be in the following format to allow for software compatibility: jpg, gif, tif, png, bmp, mov, avi, mp4 and mpg.
  5. Requests for special events should be coordinated with the program specialist to allow for planning time.
